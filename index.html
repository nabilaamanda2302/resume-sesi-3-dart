Constructor di Dart

Constructor adalah fungsi khusus dalam sebuah class yang digunakan untuk membuat objek. Constructor akan dipanggil otomatis ketika objek dibuat. Dalam Dart terdapat beberapa jenis constructor seperti default constructor, named constructor, dan constant constructor.

class Mobil { String merk; Mobil(this.merk); }

Named Constructor

Named constructor digunakan agar sebuah class dapat memiliki lebih dari satu cara untuk membuat objek. Dengan named constructor kita bisa membuat objek dari berbagai sumber data seperti Map atau JSON.

class Mahasiswa { String nama; int umur; Mahasiswa(this.nama, this.umur); Mahasiswa.dariMap(Map data) : nama = data['nama'], umur = data['umur']; }

Constant Constructor

Constant constructor digunakan untuk membuat objek yang tidak dapat diubah (immutable). Semua variabel dalam class harus menggunakan keyword final agar nilainya tidak bisa berubah setelah objek dibuat.

class Titik { final int x; final int y; const Titik(this.x, this.y); }

Enkapsulasi

Enkapsulasi adalah konsep dalam OOP yang digunakan untuk menyembunyikan data agar tidak bisa diakses secara langsung dari luar class. Di Dart variabel private biasanya ditandai dengan tanda underscore (_).

class User { String _password; User(this._password); }

Getter dan Setter

Getter digunakan untuk mengambil nilai dari variabel private, sedangkan setter digunakan untuk mengubah nilai variabel tersebut. Dengan cara ini kita bisa mengontrol bagaimana data digunakan dalam program.

class Produk { String _nama; String get nama => _nama; set nama(String value){ _nama = value; } }

Kesimpulan

Materi sesi 3 membahas berbagai jenis constructor dalam Dart serta konsep enkapsulasi yang bertujuan untuk menjaga keamanan data dalam program. Dengan menggunakan private field, getter, dan setter, data dalam program dapat lebih aman dan terkontrol.
