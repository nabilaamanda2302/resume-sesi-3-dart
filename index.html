Constructor di Dart

Constructor adalah fungsi khusus dalam sebuah class yang digunakan untuk membuat objek. Dalam Dart terdapat beberapa jenis constructor yang dapat digunakan sesuai kebutuhan program.

Named Constructor

Named constructor digunakan untuk membuat objek dengan cara yang berbeda dalam satu class. Dengan adanya named constructor, objek bisa dibuat dari berbagai sumber data seperti Map atau JSON.

Constant Constructor

Constant constructor digunakan untuk membuat objek yang tidak dapat diubah (immutable). Semua field pada class harus menggunakan keyword final.

Initializer List

Initializer list digunakan untuk menginisialisasi nilai variabel sebelum constructor dijalankan. Biasanya digunakan untuk field final atau untuk memproses nilai terlebih dahulu.

Enkapsulasi

Enkapsulasi adalah konsep dalam OOP yang digunakan untuk menyembunyikan data agar tidak dapat diakses langsung dari luar class. Tujuannya adalah menjaga keamanan dan mengontrol penggunaan data.

Getter dan Setter

Getter digunakan untuk mengambil nilai dari variabel private, sedangkan setter digunakan untuk mengubah nilai variabel tersebut serta bisa menambahkan validasi data.

Kesimpulan

Materi sesi 3 membahas berbagai jenis constructor dalam Dart serta konsep enkapsulasi yang membantu menjaga keamanan data dan mengontrol akses dalam program.
